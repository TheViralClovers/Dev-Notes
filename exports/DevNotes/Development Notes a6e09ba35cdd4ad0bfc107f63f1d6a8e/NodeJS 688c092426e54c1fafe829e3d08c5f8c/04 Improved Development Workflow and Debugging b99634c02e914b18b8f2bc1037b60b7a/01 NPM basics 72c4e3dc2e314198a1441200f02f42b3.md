# 01.NPM basics

`npm init`

We use this to initialise a npm package

This generates a `package.json` file. We can use this file to run scripts and write shorthand 

ex: `npm start` or `npm run dev`

```jsx
"scripts":{
	"start": "node app.js",
	"dev" : "nodemon app.js"
}
```
